## Databasedesign grp project
### Overview
Design and implementation of a relational database for an e-commerce platform - AltOnline. Features include hierarchical departments, products with reviews, user management, orders, and keyword-based recommendations.

## Key Components
- **ER Diagram** with recursive department hierarchy
- **Relational Schema** (normalized to 3NF)
- **SQL** table creation, population, and queries
- **Python** script for MySQL interaction via SSH tunnel
